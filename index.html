About Me

Hi, I’m Scott! I’ve spent the past decade working across education, content creation, and operations management. Along the way, I found myself increasingly drawn to solving problems with tech, so I started learning to code. I’m now building up my programming skills to automate processes, create tools, and develop projects that improve the way I work.

Projects

  • Task Tracker

    This is a simple task management app built with HTML, CSS, and JavaScript. It allows users to create, edit, and delete tasks, mark items as completed, and filter them by status. All tasks are saved in the browser using localStorage, so the list stays intact even after refreshing the page. This project helped me practice DOM manipulation, event handling, and basic data persistence.

  • Weather Lookup App

    This project is a lightweight weather search tool that fetches real-time data from an open weather API. Users can enter a city name and instantly see the current temperature, conditions, and humidity, along with relevant icons. I built this to learn how to work with external APIs, handle asynchronous JavaScript, and create a clean, responsive UI.

  • Interactive Football Game

    This project is a fun, browser-based football quiz game built with HTML, CSS, and JavaScript. Users answer multiple-choice questions, receive instant feedback, and get a final score at the end of the quiz. The app features dynamic question loading, simple animations, and a progress indicator to improve the user experience. Building this helped me practice handling user input, updating the UI with JavaScript, and structuring small interactive applications.
